# Elemental Balance (OOP Version)

Elemental Balance is an arcade-style game built in Processing where a guardian collects elemental orbs, avoids chaos hazards, and restores balance before losing all health.

## Features
- Object-oriented design using Guardian, Orb, and Particle classes
- Health system and harmful chaos orbs
- Difficulty scaling over time
- Menu, pause, victory, and game-over screens
- Particle effects and glowing visuals
- Collision detection and score tracking

## Controls
- W / S → Move up and down  
- A / D → Change speed/direction  
- P → Pause  
- R → Restart  
- I → Toggle instructions  

## Concepts Demonstrated
- Object-oriented programming (OOP)
- ArrayList data structures
- Functions and modular design
- Game state management
- Collision detection
- Event handling

## About
This project is an upgraded version of an original game built using arrays and functions, later refactored into an object-oriented structure with additional gameplay features.
